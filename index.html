HTML 테이블 편집기

HTML table 코드를 입력하고 셀을 클릭해 텍스트 수정, 정렬 변경, th와 td 변환을 한 뒤 수정된 코드를 바로 복사할 수 있는 무료 온라인 편집기입니다.

테이블 입력

HTML table 코드를 입력한 뒤 미리보기로 셀을 클릭해 수정하세요.

table, tr, td, th가 포함된 HTML 코드를 넣으면 미리보기와 함께 셀 단위 편집이 가능합니다.

테이블 미리보기

수정된 코드

사용 방법

  1. HTML table 코드를 입력합니다.
  2. 미리보기 버튼을 눌러 테이블을 렌더링합니다.
  3. 수정할 셀을 클릭해 텍스트, 정렬, th·td 변환을 진행합니다.
  4. 수정된 코드를 복사해 원하는 페이지에 붙여넣습니다.

이 도구로 할 수 있는 작업

  • HTML table 텍스트 수정
  • 좌측, 중앙, 우측 정렬 변경
  • th를 td로, td를 th로 변환
  • 수정된 결과 HTML 코드 복사

HTML table 코드를 빠르게 수정해야 할 때

이 도구는 HTML 테이블 코드를 직접 수정하기 어려운 사용자도 셀을 클릭해 직관적으로 편집할 수 있도록 만든 웹 도구입니다. 복잡한 코드 전체를 다시 작성하지 않고도 table의 내용, 정렬, 헤더 여부를 빠르게 수정할 수 있습니다.

특히 공지사항, 정보공개 문서, 소개 페이지, 표 형태의 콘텐츠를 자주 다루는 환경에서 유용합니다. HTML table 편집, th와 td 전환, 정렬 수정, 결과 코드 복사가 한 화면에서 가능합니다.

자주 묻는 질문

HTML을 몰라도 사용할 수 있나요?

네. table 코드를 붙여넣은 뒤 셀을 클릭해 수정하는 방식이라 기본 편집은 쉽게 가능합니다.

th와 td를 서로 바꿀 수 있나요?

네. 선택한 셀을 헤더 셀(th)과 일반 셀(td) 사이에서 바로 변환할 수 있습니다.

정렬도 바꿀 수 있나요?

네. 좌측, 중앙, 우측 정렬 버튼으로 선택 셀의 text-align 스타일을 바꿀 수 있습니다.
